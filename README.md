Institute Hub - A simple comprehensive web application that provides basic information about a college, including infrastructure, management, available courses, and general details.

# College Information Website

This is a basic frontend project that displays essential details about a college.

## Features
- College overview and basic details
- Information about infrastructure
- List of available courses
- Management details
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to practice and develop frontend development concepts and structuring a simple informational website.
